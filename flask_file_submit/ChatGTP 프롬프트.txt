파이썬 플라스크를 이용해서 웹사이트를 만들어줘. 조건은 다음과 같아.
1. 화면은 다음 구성요소로 이루어져있어.
1.1. 자기소개를 입력하는 텍스트상자
1.2. 파일을 1개 첨부할 수 있는 버튼
1.3. 제출 버튼
2. 서버로 전송된 파일은 다음 규칙으로 저장
2.1. 파일이 저장되는 위치는 uploads/ 이다. 이 위치에 전송된 파일과 로그파일 총 두 개의 파일이 저장될꺼야. 
2.2. 전송된 파일은 'uploads/{MD5해시}'형식으로 저장. 만약 같은 파일이 있는 경우 생략.
2.3. 로그파일은 'uploads/{MD5해시}.csv'형식으로 저장. 만약 같은 파일이 있는 경우 이어서 저장.
2.3.1. 저장할 내용(열 이름): 타임스탬프, 파일명
2.3.2. 이 파일의 내용은 'utf-8-sig' 형식으로 저장.
2.3.3. 타임스탬프는 '{yyyy-mm-dd hh:mm:ss}' 형식
2.2.4. 파일명은 사용자가 입력한 파일명 그대로 수정하지말고 저장돼어야해. txt파일에 저장되는 내용이라서 secure_filename() 함수를 사용하지마.
3. 기본적인 디렉토리 구조는 다음과 같음. 추가적인 파일이 필요하면 내용과 함께 알려줘.
    app.py
    templates/upload_form.html
    templates/submission_complete.html

4. 파일을 제출하고나면 "제출이 완료되었습니다."라는 문구가 보이는 화면으로 이동하고, 돌아가기 버튼을 보여줘.
5. 기타 요구사항
5.1. 폴더를 생성할 때에는 os.makedirs의 exist_ok 인자를 사용해서 가능하면 한줄로 만들어줘.
5.2. 한글로 주석을 생성해줘.
5.3. 자세한 설명은 생략하고 최종적으로 만들어진 소스코드만 나에게 보여줘.
5.4. 보안을 위해 입력값을 검증하고, CSRF 보호를 적용해줘. 업로드된 파일이 서버에서 실행되지 않도록 해줘.
5.4. 부트스트랩을 사용해서 최신 트랜드에 맞게 꾸며줘.
